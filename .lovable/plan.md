
## Цель

В календаре («День» и «Неделя») перейти с часовой сетки на получасовую, чтобы 30-минутные уроки корректно отображались и могли создаваться/переноситься в нужный получасовой слот.

## Изменения (только `src/components/calendar/Calendar.tsx`)

1. **Сетка слотов**: вместо массива часов `HOURS = [8..22]` ввести `SLOTS` с шагом 30 минут (`08:00, 08:30, …, 22:00` — 29 строк). Высоту слота уменьшить (`SLOT_PX = 32`), чтобы вся колонка занимала примерно столько же по высоте.

2. **DayView**:
   - Рендерить строку на каждый получасовой слот.
   - Метку времени показывать только для `:00` (чтобы интерфейс не «шумел»), `:30` — без подписи, но с более светлой разделительной линией.
   - Клик по слоту создаёт урок с этим временем (например `09:30`).
   - Drop в слот переносит урок именно на это время.

3. **WeekView**:
   - То же самое: 7 колонок × получасовые ячейки. Подпись часа — раз в час (слева).
   - Клик / drop по получасовой ячейке использует точное время.

4. **PositionedBlock**: пересчитать `top` и `height` с учётом нового `SLOT_PX` (1 час = 2 слота = 64 px). Минимальную высоту блока поставить под 30-минутный урок, чтобы он визуально занимал ровно один слот.

5. **MonthView, FAB, QuickCreateLessonSheet, DnD-логика, серверные функции** — без изменений.

## Вне рамок

- Не меняем БД, RLS, авторизацию, дизайн-токены, существующие компоненты вне `Calendar.tsx`.
- Не добавляем настройку шага слота — фиксируем 30 минут.
