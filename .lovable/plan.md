## Что я проверил

Прошёлся по основным экранам в мобильном вьюпорте (390×844): Главная, Расписание, Ученики, Финансы, Аналитика. Консоль чистая, runtime-ошибок нет, навигация работает, нижнее меню «жидкое стекло» с пилюлей — на месте, FAB «+» и тап-анимации работают.

## Что соответствует требованиям

- Жидкое стекло применено везде: карточки, шапки, нижняя навигация, листы, кнопки. Шиммер и подъём при hover/active живые.
- Старые «стрелки» у нативных `<select>` заменены на мягкий шеврон через `select { appearance: none + svg }` в `styles.css` — глобально.
- Скроллбары полностью скрыты, прокрутка пальцем/трекпадом/колёсиком работает.
- Sheet «Новый ученик» — staggered появление полей и шиммер на кнопках паттернов.
- Аналитика: переключатель `6 мес / 12 мес` — пилюля с liquid-индикатором.

## Что не соответствует и предлагаю поправить

1. **Календарь в Расписании показывает только 4 дня недели (пн–чт)**
   Видны столбцы пн 1 / вт 2 / ср 3 / чт 4, а пт/сб/вс уезжают за правый край. Просьба «пошире» не выполнена для недельного режима — горизонтальная сетка не вмещает 7 колонок на мобильном.
   Правка: в `src/components/calendar/Calendar.tsx` сделать недельную сетку 7 равных колонок на всю ширину контейнера (CSS grid `grid-cols-7`, убрать min-width у колонок, время-колонку сузить до ~40px, шрифты часов чуть меньше). Без горизонтального скролла.

2. **Переключатель «День / Нед. / Мес.» в Расписании стоит вверху справа, обычной плашкой**
   Ранее просили: «как пилюля внизу для перехода страниц жидким стеклом». Сейчас это не пилюля внизу и без liquid-индикатора.
   Правка: вынести переключатель в плавающую пилюлю над нижним меню (как `RangePill` из analytics), c `glass-strong` + `liquid-pill` бегунком и тем же easing 380ms.

3. **Date input в Финансах (`06/03/2026`) показывает нативную иконку календаря справа**
   Это та же «старомодная» иконка-стрелка по духу. На iOS/Android выглядит инородно по сравнению с остальным glass-UI.
   Правка: глобально в `styles.css` скрыть `::-webkit-calendar-picker-indicator` и `::-webkit-inner-spin-button` у `input[type="date"]`/`time`, добавить мягкую иконку календаря фоном по аналогии с тем, как сделано для `<select>`. Тап по полю продолжит открывать нативный пикер.

4. **Пилюля `6 мес / 12 мес` в аналитике — liquid-индикатор работает, но шиммер слабее, чем в нижнем меню**
   Правка: применить тот же `liquid-pill::after` shimmer с длительностью 3.2s, что уже определён в `styles.css`, и убедиться, что `.liquid-pill` класс применён к индикатору (а не только фону).
