## Цель
Сделать ассистента умнее и дать ему память, чтобы он помнил все прошлые диалоги, контекст по ученикам и принимал решения как Claude/Grok.

## Что добавлю

### 1. Долговременная память (БД)
Новая таблица `chat_messages`:
- `id`, `user_id`, `role` (user/assistant/tool), `content` (text), `tool_calls` (jsonb), `created_at`
- RLS: только владелец видит свои сообщения
- Индекс по `(user_id, created_at)`

При каждом запросе:
- Загружаю последние ~50 сообщений из БД
- Передаю целиком в модель → у ИИ есть полный контекст
- Сохраняю новое сообщение пользователя и ответ ассистента (включая вызовы инструментов)

### 2. Умнее модель и системный промпт
- Переключаю с текущей модели на **`openai/gpt-5.2`** с `reasoning: { effort: "medium" }` — это даёт рассуждения уровня Claude/Grok (многошаговое планирование, лучше работает с инструментами)
- Расширяю системный промпт: ассистент знает что он "AI-репетитор-секретарь", умеет действовать проактивно, задавать уточняющие вопросы, выполнять цепочки действий (например: "добавь Машу на вт/чт 18:00 и поставь ей урок завтра" → 3 вызова инструментов подряд)
- Контекст текущей даты, списка учеников, расписания подаётся автоматически в начале каждого запроса

### 3. UI ассистента
- Добавляю кнопку **«Очистить историю»** (удаляет сообщения из БД)
- При открытии страницы загружаются прошлые сообщения из БД (не теряются при перезагрузке)
- Индикатор "ИИ думает…" во время reasoning
